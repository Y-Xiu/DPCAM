# 2. 关键算法的描述和实现

本代码库基于ConvNeXt骨干网络，提出动态位置感知通道空间注意力模块（DPCAM）以及可学习位置信息的引入，以增强分割模型对图像空间结构和语义信息的理解能力。

## 2.1 ConvNeXt

**代码位于**: `models/convnext_unet.py` 下的 `class ConvNeXtUNet`

**描述**: ConvNeXt是一个现代化的纯卷积神经网络架构，通过引入自注意力机制的设计思想到卷积网络中，实现了与Transformer相媲美的性能。本项目采用ConvNeXt作为U-Net的骨干网络，充分利用其强大的特征提取能力和良好的迁移学习性能，为后续的注意力机制和位置信息融合奠定基础。

**实现**: ConvNeXtUNet采用编码器-解码器结构，其中编码器使用预训练的ConvNeXt骨干网络提取多尺度特征。解码器通过一系列上采样块（UpBlock）逐步恢复空间分辨率，每个上采样块支持融合来自跳跃连接的浅层特征。在解码过程中，引入灵活的注意力机制，通过注意力权重动态调整特征的重要性。最终通过全卷积层输出分割掩码，实现像素级的分类。

## 2.2 DPCAM模块

**代码位于**: `models/dpcam.py` 下的 `class DPCAM`

**描述**: DPCAM是一个改进的注意力机制模块，融合动态通道注意力、动态多尺度空间注意力和可学习位置信息的位置注意力，形成一个三维的注意力框架。该模块通过多个相互配合的子模块来捕捉特征的不同维度信息，提高模型对缺陷检测任务中关键区域的关注能力。

**实现**: DPCAM其中两个重要子模块：（1）**动态通道注意力（DynamicChannelAttention）**通过自适应平均和最大池化获取全局信息，使用MLP生成通道权重，并通过动态调制模块进一步增强权重的表达能力；（2）**动态多尺度空间注意力（DynamicSpatialAttention）**采用多个不同核大小（3×3、5×5、7×7）的卷积对空间特征进行提取，通过动态权重生成机制自适应地融合不同尺度的注意力图，使模型能够同时关注局部细节和全局上下文；这两个注意力机制在前向传播中串联应用，充分融合通道和空间维度的信息。

## 2.3 可学习的位置信息的引入

**代码位于**: `models/dpcam.py` 下的 `class PositionAttention`

**描述**: 传统的卷积神经网络缺乏对绝对位置信息的显式建模能力，这可能导致对位置相关特征的理解不足。本项目在DPCAM中引入可学习的位置编码，使网络能够显式地学习和利用图像中的位置信息作为第三个重要的子模块，特别是在工业缺陷检测中，缺陷的位置往往包含重要的语义信息。

**实现**: 位置注意力模块采用可学习的位置嵌入参数（`pos_embedding`），初始化为小的高斯分布。在前向传播时，位置嵌入根据输入特征的空间尺寸进行双线性插值调整，确保与特征图尺寸匹配。位置编码经过专用的投影层（`pos_proj`）提取位置特征，同时输入特征通过另一个投影层（`feat_proj`）进行处理。两者经过互作用层融合后，生成位置注意力权重。最后，通过element-wise乘法将位置注意力应用于原始特征，使网络能够根据学习到的位置信息动态调整特征的重要性，增强对空间位置敏感的特征学习。

# 3. 数据集与其他

为了充分验证所提出的方法的有效性和泛化性能，本代码库采用两个流行公共数据集进行工业缺陷检测。即：采用SolDef-AI数据集进行PCB微观焊点形态缺陷检测，采用VISA数据集进行PCB多类别宏观缺陷检测。数据集均按照训练集、验证集、测试集7：1.5：1.5的比例进行划分。

数据集可以通过以下方式获取：

1. **公开链接**: 两个数据集均为公开数据集，可以通过对应的官方渠道直接下载
   - SolDef-AI:(https://www.kaggle.com/datasets/mauriziocalabrese/soldef-ai-pcb-dataset-for-defect-detection)
   - VISA:(https://github.com/amazon-science/spot-diff)
